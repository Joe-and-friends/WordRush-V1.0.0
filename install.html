How to Install WordRush

Made with ❤️ by Joe & Friends
  1. Click the Download for Windows (ZIP) button on the main page.
  2. Extract the ZIP file to a folder on your computer.
  3. Open the folder and double-click WordRush.Setup.1.0.0.exe to start the installer.
  4. Follow the on-screen instructions to complete installation.
  5. Once installed, launch WordRush from your desktop or Start menu.
Note: When you run the WordRush installer, Windows may show a "Windows protected your PC" warning.
Click More info, then Run anyway to continue installing WordRush.
